## 🕶️ Что это

**DarkVPN** — коллекция рабочих прокси **VLESS + Reality** для **Throne** (Windows) и **Hiddify** (Android/iOS).  
Всё в одном файле. Импортировал — подключился — забыл о блокировках.

🔄 **Новые прокси добавляются каждые 1–5 дней.**

## ⚡ Быстрый старт

### 1. Скачай клиент
- **Windows**: [Throne](https://github.com/throneproj/Throne)
- **Android / iOS**: [Hiddify](https://github.com/hiddify/hiddify-next/releases)

### 2. Получи прокси

`curl -o proxies.txt https://raw.githubusercontent.com/web-pentest/DarkVPN/main/proxies.txt`

Или просто нажми **Code → Download ZIP**

### 3. Импортируй в клиент

**Импорт → Из файла → proxies.txt**

### 4. Подключайся

Выбери любой сервер и нажми **Connect**

---

## 📄 Формат

Каждая строка в `proxies.txt` — это рабочая `vless://` ссылка с меткой:

`vless://...@at4.darknet.run:443?...&sni=gmail.com#🇦🇹 Austria Vienna`

`vless://...@de1.example.com:443?...&sni=google.com#🇩🇪 Germany Frankfurt`

Клиент сам разберёт всё при импорте.
